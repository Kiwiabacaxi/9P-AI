Aula 02 — Hebb
Regra de Hebb (1949) — aprendizado hebbiano:
Δwi = xi · t
Δbias  = t
// passo único, sem iteração
// não converge para XOR
Aula 03 — Perceptron
Regra do Perceptron — corrige apenas erros:
y_in = bias + Σ xi·wi
y    = sign(y_in)
Δwi = α·(t−y)·xi
// só atualiza se t ≠ y
Aula 04 — MADALINE
MRII — mínimo impacto:
y_inj = biasj + Σ xi·wij
escolhe j* = argmin |y_inj|
Δwij* = α·(t−yj*)·xi
// 13 unidades ADALINE, saída OR
Aula 05 — MLP Backprop
Forward: z = w·a + b  ·  a = tanh(z)
δ saída: (t−y) · tanh'(y)
δ oculta: (Σ δ·w) · tanh'(z)
Update:  w += α · δ · a_anterior
// tanh'(x) = 1 − tanh²(x)
Aula 05 — Image Regression
Input:  (x,y) ∈ [−1,1]²
Oculta: ReLU(w·a + b)
Saída:  σ(w·a + b) → (R,G,B)
Loss:   0.5·Σ(t−y)²
// He init · SGD por pixel · 16×16
